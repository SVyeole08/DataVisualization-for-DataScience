## 📖 About

This repository documents my journey of learning **Data Visualization** for Data Science.

The goal of this repository is to understand how to effectively communicate insights from data using different visualization libraries in Python.

Rather than simply creating plots, this repository focuses on:

- 📊 Understanding different chart types
- 📈 Choosing the right visualization
- 🎨 Customizing plots
- 📉 Statistical visualization
- 📂 Working with real datasets
- 💡 Building intuition through practice

# ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/SVyeole08/DataVisualization-for-DataScience.git
```

Navigate to the project:

```bash
cd DataVisualization-for-DataScience
```

Install dependencies:

```bash
pip install matplotlib seaborn plotly cufflinks pandas numpy
```
